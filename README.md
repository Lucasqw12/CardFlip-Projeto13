# 🎴 Card Flip

Projeto de **Card Flip** desenvolvido com **HTML e CSS**, apresentando um efeito de rotação 3D ao passar o mouse sobre o cartão.
O componente exibe informações diferentes na frente e no verso, criando uma interação visual moderna e elegante.



## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3


---


---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido para praticar:

* Animações com CSS
* Transições suaves
* Criação de componentes interativos
* Efeitos 3D com CSS
